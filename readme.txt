requirement:
add jackson jar file from properties->libraries

features:
1) read data and parse text file
2) store info in arraylist and display it in the console also in a dialog box
3) converts the info into json using Jackson, creates bookInfo.json [createJson(List<book> bookInfo)]
  [json can be double checked using http://jsonlint.com/]
4) transforms json array into object using jackson [readJson()]
